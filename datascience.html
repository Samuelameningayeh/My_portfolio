  1. Financial Fraud Detection System (Python Implementation)

    Based on my co-authored research published in Elsevier


    Project Overview:
    I originally co-authored a research paper titled "A Supervised Machine Learning Algorithm for Detecting and Predicting Fraud" (Decision Analytics Journal, 2023), where we used R to demonstrate that Random Forest outperforms traditional logistic regression. For this portfolio, I re-engineered the entire experimental pipeline using Python. I migrated the logic from R to a modern Scikit-Learn and Imbalanced-Learn stack to demonstrate how these academic findings can be deployed in a production-ready environment.


    Technical Implementation:

    • Language Migration: Ported statistical models from R to Python to leverage Python's superior integration with web apps and cloud infrastructure.
    • Pipeline Design: Replaced manual R scripts with modular Python functions for preprocessing, feature selection, and model evaluation.
    • Imbalance Handling: Implemented RandomUnderSampler in Python to replicate the specific class balancing techniques used in our original study.


    Validation Results:
    The Python implementation successfully reproduced the paper's findings, achieving a 99.2% AUC score and confirming the robustness of the Random Forest architecture across different programming environments.

  2. Global COVID-19 Trend Analysis (R & Tidyverse)

    Visualizing infection rates and policy impacts using Time-Series Data


    Project Overview:
    During the onset of the global pandemic, accurate data visualization was critical for public health decision-making. In this project, I analyzed time-series data from the World Health Organization (WHO) to track the spread of the virus from its initial outbreak in Wuhan to its global expansion.


    Technical Implementation:

    • Data Wrangling (R): Utilized the dplyr and readr libraries to clean and aggregate daily case reports, handling missing values and inconsistencies in early reporting.
    • Trend Analysis: Calculated daily cumulative totals to model the exponential growth curve during the first 3 months.
    • Visualization: Engineered logarithmic scale line charts using ggplot2 to clearly display the contrast between confirmed cases and key containment events (e.g., lockdowns).


    Key Insight:
    The analysis highlighted the specific inflection points where infection rates shifted from linear to exponential growth, providing a data-driven retrospective on the timing of global intervention policies.

  3. Climate Change and Impacts in Africa


    According to the United Nations, Climate change refers to long-term shifts in temperatures and weather patterns. Such shifts can be natural, due to changes in the sun's activity or large volcanic eruptions. But since the 1800s, human activities have been the main driver of climate change, primarily due to the burning of fossil fuels like coal, oil, and gas.
    The consequences of climate change now include, among others, intense droughts, water scarcity, severe fires, rising sea levels, flooding, melting polar ice, catastrophic storms, and declining biodiversity.
