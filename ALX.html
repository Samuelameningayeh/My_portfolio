1. AI-Powered Malaria Diagnostics (Deep Learning)

Engineering a Convolutional Neural Network (CNN) for Medical Imaging


The Challenge: Malaria diagnosis traditionally relies on manual microscopic examination, which is time-consuming and prone to human error. Automation is critical for scaling healthcare in resource-limited settings.


The Solution: I built and deployed an end-to-end Computer Vision system using PyTorch. The model analyzes thin blood smear images to classify cells as "Parasitized" or "Uninfected" in real-time.


  • Architecture: Custom 4-Layer CNN (Conv2d, MaxPool, ReLU, Dropout).
  • Performance: Achieved ~95% Accuracy on the NIH Malaria Dataset.
  • Deployment: Engineered a full-stack web application using Streamlit for instant browser-based inference.
